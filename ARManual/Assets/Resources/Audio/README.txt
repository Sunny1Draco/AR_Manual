Audio Files Directory

This directory contains voice-over audio files for the AR Manual app.

File naming convention:
[device_id]_step[number]_[description].mp3

Example:
thermostat_step1_power.mp3
thermostat_step2_wifi.mp3
thermostat_step3_app.mp3

Audio specifications:
- Format: MP3
- Sample Rate: 44.1 kHz
- Channels: Mono
- Bit Rate: 128 kbps
